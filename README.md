# AMG Expertise

Site web d'expertise automobile indépendante

## Description

AMG Expertise est un site web professionnel pour un cabinet d'expertise automobile indépendant. Le site présente les services d'expertise automobile proposés aux particuliers et professionnels.

## Fonctionnalités

- **Page d'accueil** : Présentation du cabinet et de ses services
- **Page Services** : Description détaillée des différentes expertises proposées
  - Expertise avant achat
  - Expertise après sinistre
  - Expertise avant vente
  - Expertise litige
  - Diagnostic technique
- **Page À Propos** : Informations sur le cabinet, ses valeurs et sa démarche
- **Page Contact** : Formulaire de demande d'expertise et informations de contact

## Technologies

- HTML5
- CSS3
- Design responsive (mobile-friendly)

## Utilisation

Ouvrez simplement le fichier `index.html` dans un navigateur web pour visualiser le site.

## Structure du projet

```
AmgExpertise/
├── index.html      # Page d'accueil
├── services.html   # Page des services
├── about.html      # Page à propos
├── contact.html    # Page de contact
├── styles.css      # Feuille de style principale
└── README.md       # Documentation
```
